chainhash
=========

[![Build Status](https://github.com/gcash/bchd/actions/workflows/main.yml/badge.svg?branch=master)](https://github.com/gcash/bchd/actions/workflows/main.yml)
[![ISC License](http://img.shields.io/badge/license-ISC-blue.svg)](http://copyfree.org)
[![GoDoc](https://img.shields.io/badge/godoc-reference-blue.svg)](http://godoc.org/github.com/gcash/bchd/chaincfg/chainhash)
=======

chainhash provides a generic hash type and associated functions that allows the
specific hash algorithm to be abstracted.

## Installation and Updating

```bash
$ go get -u github.com/gcash/bchd/chaincfg/chainhash
```
